## Installation

```bash
npm install -g favicli
```

Or run without global install:

```bash
npx favicli
```

---

## Quick Usage

```bash
# Home menu
favicli

# Interactive setup
favicli set

# Set from a specific image
favicli set logo.png

# Detect project type
favicli detect

# Remove generated favicon files
favicli remove
```
